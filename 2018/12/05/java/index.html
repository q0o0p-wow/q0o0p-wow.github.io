数据类型转换

  • 隐式转换
    小的数据类型和大的数据类型进行混合运算的时候, 提升为大的数据类型

  • 强制数据类型转换

  • 混合运算的时候, byte,short,char不会相互转换, 小的数据类型提升为大的数据类型计算
    byte,short,char –> int –> long –> float –> double 

    8        4

    char:
    0-65535
    char可以存储中文汉字, java采用Unicode编码,每个字符占两个字节,中文也占两个字节

float:
32 位二进制位
1位符号位
8位指数位 00000000 – 11111111(-126 – 127)
float > long

任何数据类型和字符串相加都会产生新的字符串
