Java Web(mybatis-Plus)
Created 2020-05-18| Updated 2020-05-26
Post View:

mybatis plus maven包:
https://mvnrepository.com/artifact/com.baomidou/mybatis-plus-boot-starter
https://mvnrepository.com/artifact/com.baomidou/mybatis-plus

常用注解
默认连接数据库的方式: 驼峰式转换,类名的首字母小写映射到表名

  • @TableName
    映射到数据库的表名,使用此注解,类名可以和表名不一致
  • @TableId
    设置主键映射,value映射主键字段名(如果成员变量名字和主键名字不一致,需要添加此注解)
type 常见的类型介绍
NONE MP set主键,雪花算法实现
AUTO 数据库自增
INPUT 需要开发者手动输入
ASSIGN_ID MP分配ID,Long、Integer、String
ASSING_UUID 分配UUID,String

AUTO: 数据库自增,开发者不需要赋值(从最大id自增,手动赋值没用)
INPUT: 如果开发者没有手动赋值,id值会按照最大值自增
ASSIGN_UUID: 主键的类型必须是String类,自动生成UUID 进行赋值
默认生成随机数(雪花算法)

  • @TableField
