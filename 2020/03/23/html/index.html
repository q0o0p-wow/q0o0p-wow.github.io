爬虫入坑
Created 2020-03-23| Updated 2020-03-28
Post View:

网页特征： url连接地址；协议http、https；使用HTML代码

网页抓包流程:
1、 请求网址得到网页源代码
2、 在源代码中解析，获取数据
3、 如果网站中包含链接，则继续执行1~2步骤

get请求：
网址直接在地址栏里面，其中大部分请求都是get请求

post请求：
请求的时候会隐藏参数一般用于登录、注册或者填写内容提交

状态码：
1** 正在请求
2** 请求成功
3** 网站的跳转或重定向
4** 请求资源不存在
5** 服务器内部错误，
