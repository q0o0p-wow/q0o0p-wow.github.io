生成过程中一路按3次回车键就好了。第二次回车为设置密码（默认路径，默认没有密码登录）
生成成功后，去对应目录C:\Users\ASUS\ssh里面找公钥
id_rsa 是私钥
id_rsa.pub是公钥
只需复制公钥即可。
接下来为github账号配置ssh key
切换到github，在右上角找到settings，然后打开SSH keys菜单， 点击Add SSH key新增密钥，填上标题（最好跟本地仓库保持一致）。
