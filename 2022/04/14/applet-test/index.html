电脑端小程序渗透测试

Created2022-04-14|Updated2022-04-14
|Word count:247|Reading time:1min|Post View:

小程序路径：

D:\Software\WeChat\WeChat Files\Applet\wx4e8a0957bbf2ead8\151

工具

wxappUnpacker

下载:

git clone https://github.com/wanliofficial/wxappUnpacker.git

安装依赖

小程序解密包工具

下载地址:https://cowtransfer.com/s/04ec0d0a737745 或 打开【奶牛快传】cowtransfer.com 使用传输口令：7chgsl 提取；

操作流程

逆向解包分析

1. 解密

使用小程序解密工具UnpackMiniApp.exe解密，解密完成后wxpack中有解密后的.wxapkg文件

2. 解包

将解密出来的文件进行解包

解包完成后生成对应的文件

抓包分析

1. Proxifier配置

  • 配置代理服务器

profile -> proxy servers -> add.. 填入Burpsuite的代理IP和端口，选择协议HTTPS

  • 添加规则

由于系统流量较多，如果不设置指定的规则，会有许多干扰流量，不方便后续的渗透测试

proxification rules -> add.. -> wechatappex.exe,Proxy HTTPS 127.0.0.1

然后开始愉快的使用burpsuite，哈哈哈~~~
